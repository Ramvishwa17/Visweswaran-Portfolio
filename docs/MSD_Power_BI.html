Data Source

    The dataset was sourced from Kaggle and contains detailed statistics of MS Dhoni's ODI matches, including runs scored, strike rate, average, and performance against different opponents and at various venues.

Data Preparation and Transformation

1. Cleaning and Importing Data

The raw CSV file was imported into Power BI, where data cleaning steps were applied. Key tasks included:

  • Addressing missing values.
  • Formatting certain columns for analysis.

2. Calculated Columns and Measures

Calculations were created to derive key metrics, including:

  • Total Runs Scored
  • Batting Average
  • Hundreds and Fifties Count
  • Total Sixes
  • Strike Rate

3. Aggregations and Grouping

Data was aggregated to provide insights at various levels, including:

  • Year-by-year performance.
  • Performance against each opponent.
  • Performance at different grounds.

Project Objectives

  • Web scraping, Data Cleaning and form a structured data.
  • Visualize match statistics over time.
  • Provide actionable insights for team on at which stage our teams is by comparing with the opponents.

Technologies Used

The project utilized the following technologies:

  • Python for data analysis and manipulation.
  • Pandas and NumPy for data processing.
  • Matplotlib for data visualization.
